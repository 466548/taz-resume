## 職務要約

プロダクトデザイン領域に強みを持つUI/UXデザイナーとして、既存サービスの改善、新規サービスの立ち上げ、LP制作、マーケティング施策、実装支援まで一貫して担当してきました。  
課題整理や仮説立案から、要件整理、体験設計、UIデザイン、検証、改善までを一連の流れで進められることが強みです。  
また、HTML / SCSS / Ruby on Rails 環境での実装経験があり、デザインと開発の橋渡し役として、実現性を踏まえた設計や改善提案を行ってきました。  
現在はEC・決済領域のプロダクトに関わりながら、ユーザー体験と事業成果の両方を見据えた改善推進に取り組んでいます。

## 意欲・興味

- ユーザー体験と事業成果の両方を意識したプロダクト改善に興味があります
- 要件が曖昧な段階から、課題整理・体験設計・UI設計へ落とし込む役割にやりがいを感じます
- 新規立ち上げだけでなく、既存サービスの改善や運用フェーズで継続的に価値を出すことも得意です
- デザインと開発の橋渡しをしながら、チームで前に進める働き方を大切にしています
- 将来的には、プロダクトデザインを軸にしつつ、より上流の企画や改善推進にも関わっていきたいと考えています
